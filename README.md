# Uni Basvurum

Uni Basvurum is a macOS application designed to help students organize and track their university research process in one place.

Users can store information about universities, programs, application links, admission requirements, and personal notes while comparing different options efficiently.

## Features

* Create and manage university records
* Organize universities by country
* Store application and program links
* Track admission requirements (IELTS, TOEFL, etc.)
* Save personal research notes
* Mark favorite universities
* Search and filter records quickly
* Automatically fetch university logos and favicons
* Local data storage powered by SwiftData

## Technologies

* Swift
* SwiftUI
* SwiftData
* macOS
